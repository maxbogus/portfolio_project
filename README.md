Portfolio of Maxim Boguslavskiy
===============================
# Structure:
 - css:
    css used in project.
 - img:
    images converted to fit mobile, tablet and desktop.
 - Gruntfile:
    tasks performed to make img for project.
 - package.json:
    libs used by Grunt

# Update:
    To update images you have to perform two tasks:
        1. Install Grunt
        2. Run Grunt in project dir: $ grunt
